# Wikipedia 오답 문제 공통점 분석

## 오답 문제 목록
- **Q29**: Pred B, Gold I - Du Fu 시 분석 (Tang dynasty와 nomadic peoples 관계)
- **Q32**: Pred G, Gold C - Clemenceau 역사 문제 (Deutschland über alles)
- **Q34**: Pred G, Gold E - Kohlberg 도덕 발달 2단계
- **Q38**: Pred B, Gold D - 법률 문제 (절도 관련)
- **Q39**: Pred D, Gold G - 법률 문제 (장물취득)
- **Q45**: Pred E, Gold J - Singer 공리주의 반론
- **Q48**: Pred E, Gold H - 인류학 문제 (hominids 변화)

## 공통점 분석

### 1. 세부 개념 구분 문제
**특징:**
- 비슷하지만 미묘하게 다른 개념들을 정확히 구분하지 못함
- 예: Q34 - Kohlberg 1단계(처벌 회피) vs 2단계(개인적 욕구 충족)
- 예: Q38/Q39 - 법률 용어의 세부 구분 (robbery vs larceny vs receiving stolen property)

**원인:**
- Wikipedia 문서가 일반적인 개요만 제공하고 세부 구분은 충분하지 않음
- 검색된 문서가 핵심 구분 정보를 포함하지 않음

### 2. 맥락/문맥 분석 부족
**특징:**
- 질문의 맥락이나 문맥을 정확히 이해하지 못함
- 예: Q29 - 시의 내용을 통한 역사적 맥락 추론 필요
- 예: Q32 - Clemenceau의 인용문의 구체적 의미 파악 필요

**원인:**
- 단순 키워드 매칭으로는 충분하지 않음
- 시/문학 작품과 역사적 맥락을 연결하는 문서 부족

### 3. 단계/순서 관련 문제
**특징:**
- 이론의 단계나 순서를 정확히 구분하지 못함
- 예: Q34 - Preconventional level의 1단계 vs 2단계 구분
- 예: Q48 - 시간적 순서나 단계적 변화 구분

**원인:**
- Wikipedia 문서에서 단계별 세부 설명이 부족하거나 혼재
- 단계 번호나 순서 정보를 명확히 추출하지 못함

### 4. 선택지 기반 재검색 부족
**특징:**
- 선택지에 명시된 구체적 개념을 활용한 재검색이 부족
- 예: Q32 - "Deutschland über alles", "national anthem" 등 선택지 키워드 미활용
- 예: Q39 - "receiving stolen property" 선택지 키워드 미활용

**원인:**
- 질문에서만 키워드를 추출하고 선택지의 핵심 개념을 놓침
- 선택지의 구체적 용어들이 검색에 반영되지 않음

### 5. 관련성 점수 기반 필터링 한계
**특징:**
- 단순 키워드 매칭 점수만으로는 부족
- 질문의 핵심 의도와 문서의 핵심 내용이 매칭되지 않음

**원인:**
- 키워드 빈도 기반 점수가 의미적 관련성을 정확히 반영하지 못함
- 질문 유형(비교, 단계 구분, 맥락 추론)에 따른 특화 검색 부족

### 6. 다중 선택지 검증 부족
**특징:**
- 모든 선택지를 개별적으로 검증하지 않고 빠른 추론에 의존
- 예: Q45 - Singer 이론의 반론을 정확히 확인하지 않고 선택

**원인:**
- 프롬프트가 각 선택지를 개별 검증하도록 강조하지만 실행이 부족
- Wikipedia 문서에서 모든 선택지를 검증할 충분한 정보 부족

## 개선 방안

### 1. 선택지 기반 재검색 강화
- 선택지에서 구체적 개념/용어 추출
- 각 선택지의 핵심 키워드로 개별 재검색
- 예: "Deutschland über alles", "receiving stolen property", "Stage 2" 등

### 2. 단계/순서 인식 강화
- "단계", "stage", "phase" 등 단계 관련 키워드 인식
- 단계 번호와 함께 검색 (예: "Stage 2", "2단계")
- 시간 순서/진화 순서 인식

### 3. 맥락 기반 검색 강화
- 시/문학 작품의 역사적 맥락 추론
- 인용문의 구체적 의미 파악
- 작가/작품명과 역사적 사건 연결

### 4. 세부 개념 구분 강화
- 법률 용어의 정확한 정의 검색
- 이론의 단계별 차이점 검색
- 비교 대상을 명시적으로 검색

### 5. 다중 문서 검색
- 단일 문서가 아닌 관련 문서들을 함께 검색
- 예: "Kohlberg" + "Moral development" + "Stage 2"
- 예: "Tang dynasty" + "Nomadic peoples" + "Military"

### 6. 프롬프트 개선
- 각 선택지를 순서대로 검증하도록 명시
- "모든 선택지를 개별적으로 확인" 지시 강화
- 단계/순서 문제에 대한 특화 지시 추가
